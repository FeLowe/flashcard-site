• JavaScript (JS)
  • An object-oriented computer programming language commonly used to create interactive effects within web browsers.
  • Operators
  • Operators are used in logical statements to determine equality or difference between variables or values.
  • Variables
  • Variables are the way that we store values so we can use them later
  • Variable naming conventions
  • In computer programming, a naming convention is a set of rules for choosing the character sequence to be used for identifiers which denote variables, types, functions, and other entities in source code and documentation
  • Functions
  • Functions are set of statements that performs a task or calculates a value. To use a function, you must define it somewhere in the scope from which you wish to call
  • Methods
  • Method is a property containing a function definition.
  • Arguments
  • Arguments are the real values passed to (and received by) the function.
  • Return
  • The return statement stops the execution of a function and returns a value from that function.
  • Parameters
  • Parameters are the names listed in the function definition.
  • Chaining methods
  • Chaining allows us to run multiple jQuery methods (on the same element) within a single statement.
  • Strings
  • String simply stores a series of characters. A string can be any text inside quotes
  • Booleans
  • A JavaScript Boolean represents one of two values: true or false.
  • Undefined
  • Undefined means that the variable has not been declared, or has not been given a value.
  • NaN
  • The NaN property represents "Not-a-Number" value. This property indicates that a value is not a legal number. The NaN property is the same as the Number
  • Escape
  • Escape is sequences that you can include in strings to create characters that you cannot type directly. An escape sequence starts with a backslash (\) that informs the JavaScript interpreter that the next character is a special character
  • Alert
  • The alert method displays an alert box with a specified message and an OK button. An alert box is often used if you want to make sure information comes through to the user.
  • Comments
  • JavaScript comments can be used to explain JavaScript code, and to make it more readable. JavaScript comments can also be used to prevent execution, when testing alternative code.
  • jQuery
  • jQuery is a lightweight, "write less, do more", JavaScript library. The purpose of jQuery is to make it much easier to use JavaScript on your website.
  • Attributes
  • Attributes provide additional information about HTML elements.
